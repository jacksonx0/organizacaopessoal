# Sistema de Organização Pessoal

# Trabalho I 



Feito em linguagem C utilizando listas lineares simples. O objetivo é a organização de três categorias: 
* Lista de tarefas
* Lista de supermercado
* Lista de contatos



O sistema permite:
* Adicionar elementos no início da lista;
* Adicionar elementos no final da lista;
* Adicionar elementos em uma posição específica;
* Remover elementos da lista;
* Exibir os elementos armazenados.


Arquivos:
* main.c - programa principal e menu de interação; 
* lista.h - definição das estruturas e protótipos das funções; 
* lista.c - implementação das operações da lista. 


 
Para compilar o programa:
gcc main.c lista.c -o trabalho

Para executar:
./trabalho
